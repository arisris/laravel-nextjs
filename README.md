## Fullstack Laravel Next.js Starter

A skeleton project in combination laravel with next.js

Setup

```bash

git clone https://github.com/arisris/laravel-nextjs.git
cd laravel-nextjs
cd laravel && composer install
cd next pnpm install
cd ../
pnpm dev


```
